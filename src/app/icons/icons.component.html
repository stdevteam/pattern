Icons

There are two types of icons described in this library - Generated and Font Awesome.
To use the Generated icons you need to use icon-* (respective icon name).
For Font Awesome icons you need to use fa class before fa-* (respective icon name).
Please check the text color instructions here to color the icons.

Example

<!--start generated icon-->
<i class="icon-car"></i>
<!--end generated icon-->

<!--start font awesome icon-->
<i class="fa fa-backward"></i>
<!--end font awesom icon-->
