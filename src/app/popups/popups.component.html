Popup

To create a popup you need to use mat-dialog component which can be found here.

<div class="dialogStyle">
    <div class="dialogHeader flex justify-space-between align-center">
        <h4>Dialog Header</h4>
        <a class="dialogClose" (click)="dialogRef.close()"><i aria-hidden="true" class="icon-delete"></i></a>
    </div>
    <div class="dialogBody">
        <p class="text-dark">Lorem ipsum dolor sit amet, consectetur adipisicing elit.
          cum debitis dolor ducimus exercitationem impedit itaque libero odio, optio reiciendis
          soluta suscipit ut? Consequuntur cum error esse id ipsa ipsum minus odio.</p>
    </div>
    <div class="dialogMeta flex justify-space-between align-center">
        <button class="btn white" (click)="dialogRef.close()">Cancel</button>
        <button class="btn green" (click)="dialogRef.close()">Done</button>
    </div>
</div>
