Grid System

Grid Sizes

Grid system allows up to 12 columns across the page.
If you do not want to use all 12 columns individually, you can group the columns together to create wider columns. To create columns you need to add row class to the parent element and from col-1 to col-12 classes to child elements. The sum of child element col classes should not exceed 12.
