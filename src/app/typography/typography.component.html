Typography

Headings

Headings are defined with the <h1> to <h6> tags. <h1> defines the most important heading. <h6> defines the least important heading. You need to add the respective tags only, no classes are needed.

This is an h1 Heading 30px

This is an h2 Heading 24px

This is an h3 Heading 20px

This is an h4 Heading 18px

This is an h5 Heading 16px
This is an h6 Heading 14px
<h1>This is an h1 Heading</h1>
<h2>This is an h2 Heading</h2>
<h3>This is an h3 Heading</h3>
<h4>This is an h4 Heading</h4>
<h5>This is an h5 Heading</h5>
<h6>This is an h6 Heading</h6>

Paragraphs

Paragraphs are defined with <p> tag. There are different sizes of paragraphs which are defined by respective classes from font-xs to font-xxxl.

This is a paragraph. 28px

This is a paragraph. 24px

This is a paragraph. 22px

This is a paragraph. 20px

This is a paragraph. 18px

This is a paragraph. 16px

This is a paragraph. 14px

This is a paragraph. 12px

<p class="font-xxxl">This is a paragraph.</p>
<p class="font-xxl">This is a paragraph.</p>
<p class="font-xl">This is a paragraph.</p>
<p class="font-lg">This is a paragraph.</p>
<p class="font-md">This is a paragraph.</p>
<p class="font-normal">This is a paragraph.</p>
<p class="font-sm">This is a paragraph.</p>
<p class="font-xs">This is a paragraph.</p>

Text colors

To color the texts in paragraphs/headings/icons/hyperlinks you need to use the following classes.

This text is dark.

This text is grey.

This text is red.

This text is green.

This text is blue.

<p class="text-dark">This text is dark.</p>
<p class="text-grey">This text is grey.</p>
<p class="text-red">This text is red.</p>
<p class="text-green">This text is green.</p>
<p class="text-blue">This text is blue.</p>
<p class="text-blue">This text is white.</p>

Text alignment

You can align the text using the respective class.

This is a left aligned text.

This is a centered aligned text.

This is a right aligned text.

<p class="text-left">This is a left aligned text</p>
<p class="text-center">This is a centered aligned text</p>
<p class="text-right">This is a right aligned text</p>

Text transform

You can transform the text in components with text capitalization classes.

lowercased text.

uppercased text.

capitalized text.

<p class="lowercase">lowercased text.</p>
<p class="uppercase">uppercased text.</p>
<p class="capitalize">capitalized text.</p>
