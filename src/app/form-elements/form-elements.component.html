Form elements

Buttons

There are several types of buttons described in this library. To use them you need to add btn class before the respective button name (e.g. white/blue/disabled/etc.) 

<button class="btn white">white</button>
<button class="btn grey">grey</button>
<button class="btn blue">blue</button>
<button class="btn green">green</button>
<button class="btn disabled">disabled</button>

Buttons can be used with "+" character from the left side.

<button class="btn green  leftplus"><span>+</span>with plus</button>

To use the fullwidth buttons you need to add fullWidth class to the other classes 

<button class="btn blue fullWidth">full width</button>

Button Groups

You can group a series of buttons together (on a single line) in a button group with the needed buttons.

<ul class="flex">
    <li class="flex">
        <button class="nav-btn" disabled><i class="fa fa-backward" aria-hidden="true"></i></button>
        <button class="nav-btn" disabled><i class="fa fa-play fa-flip-horizontal" aria-hidden="true"></i></button>
    </li>
    <li class="flex">
        <button class="nav-btn"><i class="fa fa-play" aria-hidden="true"></i></button>
        <button class="nav-btn"><i class="fa fa-forward" aria-hidden="true"></i></button>
    </li>
</ul>

Selects

To create a select field you need to use mat-select component which can be found here.

option 1 option 2 option 3

  <mat-option [value]="1">option 1</mat-option>

  <mat-option [value]="3">option 3</mat-option>

Inputs

There are two types of inputs described in this library - Simple and Quantity inputs. 

<!--start simple input-->
<div class="field">
    <label class="text-grey block">Simple input:</label>
    <input type="text" placeholder="placeholder">
</div>
<!--end simple input-->

<!--start quantity input-->
<div class="field">
    <label class="text-grey block">Quantity input:</label>
    <input type="text" placeholder="10 Minutes">
    <div class="quantity-nav">
        <div class="quantity-button quantity-up">
            <i aria-hidden="true" class="fa fa-angle-up"></i>
        </div>
        <div class="quantity-button quantity-down">
            <i aria-hidden="true" class="fa fa-angle-down"></i>
        </div>
    </div>
</div>
<!--end quantity input-->

Checkboxes

Checkboxes are used if you want the user to select any number of options from a list of preset options. To create a checkbox you need to add checkbox class to <div>. Here are described the enabled/disabled checkboxes.
Add the disabled attribute to the <input> and the custom indicator and label description. Add the checked attribute to the <input> and the custom indicator and label description.

<!--start custom checkbox-->
<div class="checkbox">
    <label class="flex">
        <input type="checkbox">
        <span class="checkIcon"><i class="fa fa-check"></i></span>
        <span class="label">custom checkbox</span>
    </label>
</div>
<!--end custom checkbox-->

<!--start checked checkbox-->
<div class="checkbox">
    <label class="flex">
        <input type="checkbox" checked>
        <span class="checkIcon"><i class="fa fa-check"></i></span>
        <span class="label">checked checkbox</span>
    </label>
</div>
<!--end checked checkbox-->

<!--start disabled checkbox-->
<div class="checkbox">
    <label class="flex">
        <input type="checkbox" disabled>
        <span class="checkIcon"><i class="fa fa-check"></i></span>
        <span class="label">disabled checkbox</span>
    </label>
</div>
<!--end disabled checkbox-->

Radios

To create a radiobutton you need to add radioBtn class to <div>.To select the radiobutton you need to add the checked attribute to the <input> and the custom indicator and label description.

<!--start custom radio-->
<div class="radioBtn" (click)="onChangeMinutes(5)">
    <input type="radio" id="customRadio" name="time">
    <label for="customRadio">custom radio</label>
</div>
<!--end custom radio-->

<!--start checked radio-->
<div class="radioBtn" (click)="onChangeMinutes(5)">
    <input type="radio" id="checked" name="time" checked>
    <label for="checked">checked radio</label>
</div>
<!--end checked radio-->

Switch

To create a switch (on/off) you need to add switch class to div.

<!--start switch-->
<div class="switch">
    <input id="switch" type="checkbox" checked/>
    <label for="switch"></label>
</div>
<!--end switch-->
